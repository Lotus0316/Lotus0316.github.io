Linux基本操作

发表于2024-05-05|更新于2024-05-05|Linux

Linux常用命令

Screen窗口

  1. 使用screen -S name创建一个名为name的screen
  2. 使用screen -ls来看那些screen正在被终端链接（Attached的那个）
  3. 使用screen -R name来连接到名为name的screen，若在screen不存在则自动创建
  4. 使用Ctrl+a+d将会话移动回后台
  5. 使用kill+pid来关闭某个screen，其中pid在-ls中查看

Zip压缩

  1. 在父级目录使用zip -r name.zip filename或者在任意位置使用 /path/to/filename来压缩文件夹
  2. 使用unzip name,zip来解压缩

查看

  1. 使用nano查看并编辑文件
  2. 使用pwd来查看当前路径
  3. 使用ls来查看目录下文件，使用ls -la来查看包括隐藏文件的所有文件

安装

  1. yum -y update检查yum安装工具
  2. 使用管理员权限的sudo yum -y install appname来安装软件

用户

  1. 使用useradd -m username创建用户
  2. 使用su - username切换用户，使用su - 回到root用户

发送

  1. 使用sudo yum -y install lrzsz安装lrzsz包
  2. 使用sz filename来从服务器下载，使用rz来上传

文件

  1. 新建文件夹：mkdir name
  2. 删除文件夹：rm -r name
  3. 新建文件：nano filename
  4. 删除文件：rm filename注意文件一定要带上扩展名
